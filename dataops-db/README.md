# DataOps DB (Fabric SQL Database)

This folder contains the source-controlled database objects for the DataOps/Control framework used to drive ingestion and logging in Microsoft Fabric.

## MVP Objects
- ref.SourceSystem
- cfg.Dataset
- dc.DataContract
- log.IngestRun
- dbo.vw_ActiveDatasets

## Notes
- Database objects live in the Fabric-managed SQL project folder (generated by Fabric Git integration).
- Optional human scripts (seed data, ad hoc queries) can live under /scripts.
